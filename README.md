# quicksmoothie
Plot device motion and gyro live using [smoothie charts](http://smoothiecharts.org/) and [gyro.js](https://github.com/tomgco/gyro.js).
## Packages:
- node.js
- express 
- socket.io
- gyro.js
- smoothie charts


## Aim: 
learn what signals from accelerometer cna be used for controls.
